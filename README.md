# CommitMentIssues

**CommitMentIssues** is a tool for analyzing GitHub repositories to support HR and technical recruitment processes.

This tool evaluates repository quality through metrics such as:

- commit message consistency  
- alignment between commits and actual code changes  
- project structure and complexity  
- adherence to coding standards  

The project is currently in an early development stage.
